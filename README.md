# EduConnect

A web-based student–teacher interaction platform for doubt discussion, notes sharing, and assignment management.

## Features
- 🔐 Role-based login (Student / Teacher)
- 💬 Doubts board — post and delete questions
- 📄 Notes — add, edit, delete with timestamps
- 📌 Assignments — add tasks, toggle done/pending, delete
- 🎓 Classroom — create/join via code, member list, attendance with Present/Absent toggle
- ⚙️ Settings — update name/role, clear all data
- 🌙 Dark mode (persisted)
- 🔔 Toast notifications instead of alert() dialogs

## How to Run
Open `index.html` in any modern browser. No server required.

## Tech Stack
- Vanilla HTML, CSS, JavaScript
- localStorage for data persistence
- DM Sans + DM Mono fonts (Google Fonts)

## Notes
This is a frontend-only demo. For a production app, replace localStorage with a real backend (Firebase, Supabase, or a Node.js/Express API).
